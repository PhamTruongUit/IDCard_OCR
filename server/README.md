### Setting server:

#### Install

- with python <= 3.8.x
- using terminal
- create virtualenv
  ```
  cd server
  python -m venv myenv
  myenv/Scripts/activate
  ```
- install require
  ```
  python -m pip install -- upgrade pip
  pip install -r requirements.txt
  ```

#### Run

- using terminal
- using virtualenv
  ```
  python main.py
  ```
